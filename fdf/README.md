# FdF (Fil de Fer)

A 3D wireframe relief map renderer using MiniLibX.

## Description
FdF is a graphical project that takes a coordinates map as input and renders it in 3D using isometric projection. It involves basic computer graphics concepts, window management, and event handling.

## Usage
Compile the project:
```bash
make
```
Run it with a .fdf map:
```bash
./fdf [map.fdf]
```
