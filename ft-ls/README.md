# ft-ls

A custom implementation of the ls command.

## Description
Recoding the `ls` command, focusing on directory stream management and file metadata. It handles various flags like `-l`, `-R`, `-a`, `-r`, and `-t`, along with proper sorting by name or modification time.

## Usage
Compile the project:
```bash
make
```
Run it:
```bash
./ft_ls [options] [path]
```
